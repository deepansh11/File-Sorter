# File-Sorter
<p>A simple file sorting script that allows you to keep your "Downloads" folder or whichever folder you choose clean, pre-defined extensions include but not limited to:- </p>
<ul>
  <li>pdf</li>
  <li>png</li>
  <li>jpg</li>
  <li>jpeg</li>
  <li>mp4</li>
  <li>txt</li>
  <li>pptx</li>
  <li>docx</li>
  <li>xlsx</li>
</ul>
<br>
# Installation
`pip install -r requirements.txt`
